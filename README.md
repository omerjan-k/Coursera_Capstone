# Coursera Capstone

Short description
- This project aims for Applied Data Science Capstone. Jupyter Notebook(s) demonstrating data exploration, modeling, and results.

Repository metadata
- Repository: omerjan-k/Coursera_Capstone
- Description: This project aims for Applied Data Science Capstone.
- Languages: 100% Jupyter Notebook

Quick start
- Clone: git clone https://github.com/omerjan-k/Coursera_Capstone.git
- (Optional) Create a venv: python -m venv venv && source venv/bin/activate
- Install dependencies: pip install -r requirements.txt  (or pip install pandas numpy matplotlib seaborn scikit-learn jupyter)
- Run: jupyter notebook or jupyter lab and open the .ipynb file(s)

Repository layout
- *.ipynb — main notebook(s)
- data/ — optional datasets used by notebooks
- requirements.txt — dependencies (create if missing)
- images/ — optional figures

Notes
- Do not commit secrets or API keys. Use environment variables or a local config excluded by .gitignore.
- To reproduce results, set random seeds in notebooks and pin package versions.
- For Colab: File → Open notebook → GitHub and paste the repo URL.
